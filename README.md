The Task Management System is a JavaFX-based desktop application designed to 
help users manage and organize tasks efficiently. The system allows users to add, 
view, and analyze tasks using a structured and user-friendly graphical interface. 
When the user opens the application, all existing tasks are loaded from a CSV file 
and displayed in a ListView. Each task appears with its details, allowing the user to 
easily understand its information and status. 
The user can add new tasks manually through a form by entering the required 
fields such as title, status, added by, and creation date. The system validates all 
inputs before saving the task to ensure data correctness. 
In addition, the system provides advanced features that allow the user to analyze 
tasks. For example, the user can view tasks added by a specific person, identify the 
most active contributor, count open and closed tasks, and filter tasks based on 
specific conditions. 
The application also includes a menu bar that allows the user to customize the 
appearance and view information about the system.
